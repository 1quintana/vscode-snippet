# vscode-snippet

![Preview](preview.gif)

A VisualStudioCode extension for [cht.sh](https://cht.sh/).

## Installation

[**Click here to install the extension**](vscode:extension/vscode-snippet.Snippet)

## Usage

1. Hit `Cmd` + `Shift` + `P`
2. Run `Snippet: Find`.
3. Type your query and hit enter.
